# Supermemory

> The memory layer for AI agents. Context engineering platform powering enterprise APIs, developer plugins, and a personal app that remembers everything.

Supermemory is a research and product lab building the best context and memory infrastructure for AI agents.

## Core API Endpoints

- Add documents: POST https://api.supermemory.ai/v3/documents
- Search memories: POST https://api.supermemory.ai/v3/search
- Full reference: https://docs.supermemory.ai/api-reference

## Key Features

- **User Profiles**: Builds deep user profiles from behavior — intent, preferences, and context
- **Memory Graph**: Custom vector graph engine with ontology-aware edges
- **Retrieval**: Hybrid vector + keyword search with sub-300ms latency
- **Extractors**: Understands any format — PDFs, web pages, images, audio
- **Connectors**: Notion, Slack, Google Drive, S3, Gmail, and custom sources
