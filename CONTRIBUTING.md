# Contributing

## Setup

```bash
bun install
bun run check
```

## Development

- Use `bun run typecheck` for TypeScript checks.
- Use `bun run lint` for Biome checks.
- Use `bun run test` for the test suite.
- Use `bun run smoke` for a quick CLI sanity check.

Keep changes focused and include tests for crawler, path mapping, CLI, or writer behavior when those areas change.
